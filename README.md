# 📅 Interactive Wall Calendar Component

## 🚀 Overview

This project is a **fully responsive interactive calendar component** built using **React and Next.js**, inspired by a physical wall calendar design. It combines a visually appealing UI with functional features like date range selection and an integrated notes system.

---

## 🎯 Features

### 📅 Date Range Selection

* Select a **start date** and **end date**
* Selected range is visually highlighted
* Clear distinction between start, end, and in-between dates

### 📝 Integrated Notes System

* Add **general notes** for the month
* Add **date-specific notes** for selected dates
* Easily switch between general and date-based notes

### 💾 Data Persistence

* Notes are stored using **localStorage**
* Data persists even after page refresh

### 🎨 Wall Calendar UI

* Hero image with modern design
* Diagonal overlay for aesthetic appeal
* Clean and minimal layout

### 📱 Fully Responsive Design

* **Desktop:** Side-by-side layout (calendar + notes)
* **Mobile:** Stacked layout for better usability
* Optimized for touch interactions

---

## 🛠 Tech Stack

* **React.js**
* **Next.js**
* **Tailwind CSS**

## ▶️ How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/your-username/calendar-project.git
```

2. Navigate to project folder:

```bash
cd calendar-project
```

3. Install dependencies:

```bash
npm install
```

4. Run the development server:

```bash
npm run dev
```

5. Open in browser:

```
http://localhost:3000
```
